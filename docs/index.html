What is this?

A curated collection of 600+ free games that are updated daily: puzzle games, word games, trivia games, logic games, geography games and more! All games are completely free and do not require an account or subscription.

What is a dle?

A dle, or daily game, is generally a game that changes everyday which has the same version for everyone that plays, often taking inspiration from Wordle.

Can I help?

Sure! You can suggest a dle, report a bug, or give general feedback. If you want to support monetarily, you can donate on Ko-fi. Lastly, you can look at the code on GitHub and propose changes, as this project is open source.
